# 回路図・配線詳細

ブレッドボード試作〜基板化まで共通で使う配線仕様。ピン番号は Arduino Nano Every の表記。
KiCad 化（Phase 4）まではこのドキュメントが回路のマスターとなる。

## 1. 電源部

```
単3×8 (12V, 電池ボックスの内蔵スイッチを電源SWとして使用)
  │
  ├─[+12V レール]──┬── 7セグ セグメント電源（330Ω 経由 → 各セグメント）
  │                ├── 2SA1015 エミッタ ×3（桁コモン用ハイサイド）
  │                ├── TBD62083 COM 端子 (pin10)
  │                ├── 2SA1015 ベースプルアップ 47kΩ ×3
  │                └── Nano Every VIN（内蔵 DCDC で 5V 生成、7〜21V 入力可）
  │
  └─[GND]────────── 全回路共通
+12V レールに 100µF + 0.1µF、Nano Every の 5V ピン直近に 100µF + 0.1µF を入れる。

[+5V レール]（Nano Every の 5V ピンから供給）
  ├── LCD (VDD, バックライト)
  ├── GPS GT-502MGG-N (VCC)          … 消費 ~30mA
  ├── DS1307 RTC モジュール
  ├── AE-BM1422AGMV (VCC, 5V直結可)
  ├── 反転バッファのプルアップ 1kΩ ×2
  └── ボタンラダーの上側
```

- Nano Every の 5V 供給能力は内蔵 DCDC(MPM3610)由来で余裕あり（合計 ~150mA 程度の想定負荷）。
- 7セグの電流は 12V レールから直接流れ、5V 系を通らない。

## 2. GPS（GT-502MGG-N）

出力が 3.3V LVTTL（実測 H≈2.8V）で、5V 動作の ATmega4809 の Vih(0.7×VDD=3.5V) に届かないため、
TX と 1PPS は 2SC1815 の反転バッファで受け、ファームウェア側で論理を戻す。

```
GPS TX ──1kΩ──┬─ B                    GPS 1PPS ──1kΩ──┬─ B
              Q1 2SC1815                              Q2 2SC1815
   E ─ GND    C ──┬── D0 (RX1)            E ─ GND     C ──┬── D2
                  └─ 1kΩ ─ +5V                            └─ 1kΩ ─ +5V
```

- D0(RX1)=PC5 は信号が反転して届くため、`PORTC.PIN5CTRL |= PORT_INVEN_bm` で論理反転して受信する。
- 1PPS は反転しているので **FALLING エッジ**の外部割り込みが本来の秒頭に対応する。
- MCU → GPS 方向（設定コマンド用、任意）:

```
D1 (TX1) ──1kΩ──┬── GPS RX
                └── 1kΩ ─ 1kΩ ── GND   （1:2 分圧で 5V → 3.3V）
```

- GPS の電源は +5V（許容 3.0〜5.5V）。9600bps / NMEA0183 / 1Hz（デフォルトのまま使用）。
- 配線色（先バラ品の一般例。**必ず現物のデータシートで確認**）: 電源/GND/TX/RX/1PPS の 5 本を使用。

## 3. I2C バス（A4=SDA / A5=SCL、5V バス）

```
A4 (SDA) ──┬── DS1307モジュール SDA ──┬── AE-BM1422AGMV SDA
A5 (SCL) ──┴── DS1307モジュール SCL ──┴── AE-BM1422AGMV SCL
```

- DS1307（アドレス 0x68）はモジュールキットにプルアップあり（組み立て時に実装）。
- AE-BM1422AGMV（アドレス 0x0E）はレベル変換・レギュレータ内蔵、5V 直結。
- プルアップ合計が強すぎ／弱すぎないか試作時に波形または動作で確認。無ければ 2kΩ（1kΩ×2 直列）を 5V へ。
- **地磁気モジュールの取付**: 12V 配線・電池・7セグ・スピーカー等の磁気源から極力離す。
  基板化の際はセンサーだけ端に配置。取付方位のズレはファームの取付オフセット定数で補正。

## 5. 7セグ 3 桁ダイナミック点灯

A-2301SRD-A（2.3インチ・アノードコモン・セグメント Vf≈7.6V）×3 桁。
セグメント側（カソード）は 3 桁で共通バス化し TBD62083 でシンク、
桁コモン（アノード）は 2SA1015 のハイサイドスイッチで 12V から桁ごとに給電する。

### セグメント側（1 系統 × 7、DP は未使用）

```
+12V →(桁スイッチ経由)→ 桁コモンアノード → [セグメント a のLED] → 桁1〜3のカソードを共通バス化
                                                │
                                     330Ω ── TBD62083 OUTn ─(内部)─ GND
                                                          INn ← MCU セグメントピン (H=点灯)
```

| セグ | MCU ピン | TBD62083 (DIP18) |
|---|---|---|
| a | D12 | IN1(pin1) → OUT1(pin18) |
| b | D13 | IN2(pin2) → OUT2(pin17) |
| c | A0 | IN3(pin3) → OUT3(pin16) |
| d | A1 | IN4(pin4) → OUT4(pin15) |
| e | A2 | IN5(pin5) → OUT5(pin14) |
| f | A3 | IN6(pin6) → OUT6(pin13) |
| g | A6 | IN7(pin7) → OUT7(pin12) |
| — | — | IN8 は GND に接続（未使用入力の開放禁止） |

- TBD62083: pin9 = GND、pin10 = COM → +12V。
- セグメント電流 ≈ (12 − 7.6 − 0.2 − 0.3)V ÷ 330Ω ≈ **12mA**（1/3 デューティ）。
- 7セグ表示器のピン→セグメント対応は A-2301SRD-A データシートを見て結線（桁ごとに同じ配線を 3 回繰り返し、
  カソードを共通バスに落とす。**コモンアノードピンだけ桁ごとに分ける**）。

### 桁select側（×3 回路、DIG1=左桁）

```
                +12V
                 │ E
        47kΩ   2SA1015 (PNP)
   +12V ─┴─ B ──┤
                 │ C ────────── 桁 n のコモンアノード
        1kΩ
   B ────┴────── C  2SC1815 (NPN)
   │             E ─ GND
   1kΩ
   │
  MCU DIGn (D9/D10/D11, H=桁ON)
```

- MCU H → 2SC1815 ON → 2SA1015 ベースが 1kΩ 経由で引かれ ON → 桁に 12V 供給。
- 桁ピーク電流 ≈ 7 セグ × 12mA ≈ 84mA（2SA1015 の Ic max 150mA に対し余裕）。
- 47kΩ は 2SA1015 の確実なオフ用（B–E 間プルアップ）。
- 桁切替はタイマ割り込みで 1ms/桁（全体 333Hz リフレッシュ）。

## 6. ボタン（抵抗ラダー → A7）

```
+5V ──┬─[MODE]──────────────┬── A7 ──┬── 47kΩ ── GND
      ├─[BTN A]── 47kΩ ─────┤        └── 0.1µF ── GND（ノイズ対策）
      └─[BTN B]── 47kΩ ── 47kΩ ──┘
```

| ボタン | 理論 ADC 値 (10bit) | 判定しきい値（初期値、実測で調整） |
|---|---|---|
| なし | ≈0 | < 150 |
| MODE | ≈1023 | > 800 |
| A | ≈512 | 430–650 |
| B | ≈341 | 250–430 |

同時押しは非対応（最も 5V に近い経路が勝つ）。

## 8. ブレッドボード配置案

- **EIC-801**: Nano Every、反転バッファ ×2、I2C 取り回し、ボタンラダー
- **BB-102 ×3**: 7セグ 1 桁 + 桁スイッチ（2SC1815+2SA1015）を各 1 枚に。セグメントバスは 3 枚を渡り配線
- TBD62083 は EIC-801 か BB-102 の 1 枚目に
- LCD・GPS・RTC・地磁気はジャンパで引き出し（地磁気は 12V 配線から離す）
